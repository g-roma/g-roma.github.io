Freesound Radio

Freesound Radio was my first shot at an online collaborative interface for music creation, allowing users to create small compositions (sample patches) using Freesound sounds. In version 2, sample patches could be nested in other patches. It was online from 2008 to 2012. The client side was entirely coded in ActionScript, as Flash was the only option for audio programming in the web. With the consolidation of HTML5, Flash started dying, and the Web Audio API replaced it for audio development.
