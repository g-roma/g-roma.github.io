SuperColliderAU

SuperColliderAU was first presented in 2006, allowing to wrap SuperCollider synth definitions into AudioUnits plug-ins. Two main modes were supported: in interactive mode, a SuperCollider server instance wrapped in a generic AudioUnits plug-in can receive OSC commands from any client (typically the SuperCollider language). In standalone mode, a synth definition and all necessary metadata is bundled into the plug-in, which can be controlled using standard widgets (typically sliders) defined for each control.
